# 页面路由规划

---

## 用户端（TabBar 页面）

| 路径 | 页面 | 说明 |
|------|------|------|
| /pages/index/index | 首页 | 轮播图、分类入口、推荐商品 |
| /pages/menu/menu | 菜单 | 分类导航、商品列表、搜索 |
| /pages/cart/cart | 购物车 | 商品列表、数量修改、结算 |
| /pages/order/order | 订单 | 订单列表、状态筛选 |
| /pages/user/user | 我的 | 用户信息、地址管理、历史记录 |

### 首页 (index)
- **主要功能**：轮播图展示、分类快捷入口、推荐/热销商品、进入菜单
- **需要数据**：banners、categories（前几名）、products（热销/推荐，按 sales 或 tag）

### 菜单 (menu)
- **主要功能**：左侧分类导航、右侧商品列表、搜索入口、加入购物车
- **需要数据**：categories、products（按 categoryId 分页）

### 购物车 (cart)
- **主要功能**：展示购物车项、修改数量、删除商品、全选、结算
- **需要数据**：cart（items、totalCount、totalAmount）

### 订单 (order)
- **主要功能**：Tab 切换状态（全部/待付款/待发货/已完成）、订单列表、去支付/取消
- **需要数据**：orders（按 status、page、limit）

### 我的 (user)
- **主要功能**：头像昵称、收货地址、订单统计入口、设置入口、退出登录
- **需要数据**：userInfo（nickname、avatar、addresses）、订单数量统计

## 用户端（子页面）

| 路径 | 页面 | 说明 |
|------|------|------|
| /pages/product-detail/product-detail | 商品详情 | 轮播图、规格选择、加入购物车 |
| /pages/order-confirm/order-confirm | 订单确认 | 地址选择、商品确认、支付 |
| /pages/order-detail/order-detail | 订单详情 | 订单信息、物流追踪 |
| /pages/address-list/address-list | 地址列表 | 增删改查收货地址 |
| /pages/address-edit/address-edit | 编辑地址 | 表单填写 |
| /pages/search/search | 搜索 | 搜索框、历史记录、热门搜索 |

### 商品详情 (product-detail)
- **主要功能**：图片轮播、规格选择、数量增减、加入购物车、立即购买
- **需要数据**：product（id 入参）、categories（可选，面包屑）

### 订单确认 (order-confirm)
- **主要功能**：选择收货地址、展示商品、填写备注、选择配送/堂食、发起支付
- **需要数据**：addresses、cart 或 items（来自商品详情或购物车）

### 订单详情 (order-detail)
- **主要功能**：订单状态、商品明细、配送地址、支付信息、物流、取消/去支付
- **需要数据**：order（id 入参）

### 地址列表 (address-list)
- **主要功能**：展示收货地址、新增、编辑、删除、设默认
- **需要数据**：addresses（来自 userInfo）

### 编辑地址 (address-edit)
- **主要功能**：省市区选择、详细地址、姓名、手机号、是否默认、保存
- **需要数据**：无（新增）或 address（编辑时 id 入参）

### 搜索 (search)
- **主要功能**：输入搜索、历史记录、热门搜索、搜索结果列表
- **需要数据**：searchHistory（本地）、hotKeywords（本地或接口）、products（keyword + page）

## 店长端（分包 pages_admin）

| 路径 | 页面 | 说明 |
|------|------|------|
| /pages_admin/dashboard/dashboard | 数据看板 | 营收统计、图表展示 |
| /pages_admin/products/products | 商品管理 | 列表、搜索、筛选 |
| /pages_admin/product-edit/product-edit | 编辑商品 | 表单、图片上传 |
| /pages_admin/categories/categories | 分类管理 | 列表、排序 |
| /pages_admin/orders/orders | 订单管理 | 列表、筛选、操作 |
| /pages_admin/order-detail/order-detail | 订单详情 | 订单信息、操作按钮 |
| /pages_admin/settings/settings | 设置 | 店铺信息、营业时间 |

### 数据看板 (dashboard)
- **主要功能**：今日/本周/本月概览、销售趋势图、商品排行、分类占比
- **需要数据**：statistics/overview、statistics/sales、statistics/products、statistics/categories

### 商品管理 (products)
- **主要功能**：商品列表、按分类筛选、搜索、上架/下架、编辑、新增
- **需要数据**：admin/products（page、limit、categoryId、keyword）、categories

### 编辑商品 (product-edit)
- **主要功能**：基本信息、多图上传、规格配置、价格库存、保存/发布
- **需要数据**：product（编辑时 id 入参）、categories

### 分类管理 (categories)
- **主要功能**：分类列表、拖拽排序、启用/禁用、编辑、新增
- **需要数据**：admin/categories

### 订单管理 (orders)
- **主要功能**：订单列表、按状态筛选、订单号搜索、发货、导出
- **需要数据**：admin/orders（page、status、orderNo）

### 订单详情 (order-detail)
- **主要功能**：订单明细、用户信息、地址、物流、发货/完成操作
- **需要数据**：admin/orders/:id

### 设置 (settings)
- **主要功能**：店铺名称、Logo、营业时间、联系电话、通知设置
- **需要数据**：shopConfig（店铺配置接口）
